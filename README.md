# 分贝通客户商机雷达

> 分贝通销售专属客户动态监控与商机发现工具，适配飞书 Aily 智能伙伴，集成 KAMI 排版系统。

## 这是什么

一个帮你自动监控客户公司动态、识别支出管理商机、生成顾问式跟进话术，并输出印刷级专业 PDF 的 Aily Skill。

**解决的问题**：销售每天花大量时间刷新闻找商机，但经常漏掉关键动态，或者看到了也不知道怎么自然地切入。这个 Skill 帮你把"监控→评估→话术→排版"全流程自动化，最终输出可直接交付的专业级 PDF。

## 核心能力

| 能力 | 说明 |
|------|------|
| 动态监控 | 自动搜索客户融资、出海、人事变动等关键动态，严格过滤旧闻 |
| 商机评估 | 基于支出管理痛点评估商业价值，判断差旅量、出海需求、合规压力 |
| 话术生成 | 生成顾问式跟进话术，有判断、有数据、有时机感，不像群发 |
| 质检验收 | 独立质检话术质量，杜绝AI味儿、宣传腔、模板感 |
| **专业排版** | **应用 KAMI 排版系统，生成印刷级 PDF，暖米色底色 + 油墨蓝强调色** |

## 架构

采用 PaperClip 公司化分工模型，5 个专业化环节流水线作业：

```
Researcher (研究员) → Analyst (分析师) → Writer (话术撰写员) → Evaluator (质检员) → Formatter (排版员)
    搜索+去重            商机评估+排序        套用模板生成话术        独立验收输出质量         应用KAMI排版生成PDF
```

## 话术风格

融合 [人味儿写作心法](https://github.com/orange2ai/renwei-writing)，追求**顾问式销售**而非朋友闲聊或AI群发：

| 风格 | 示例 | 问题 |
|------|------|------|
| AI模板腔 | "恭喜完成B轮融资，我们提供一站式解决方案" | 没有人，客户3秒划走 |
| 朋友闲聊 | "厉害了！差旅这块肯定要头疼" | 没有专业度 |
| **顾问式（目标）** | **"B轮2亿，按行业经验，6个月内团队会扩到300人+，差旅支出至少翻3倍"** | **有判断、有数据、客户想听下去** |

## 使用方式

### 安装到飞书 Aily 智能伙伴

1. **上传本 Skill**：将 `SKILL.md` 和 `references/` 目录上传到你的 Aily 智能伙伴

2. **首次触发自动引导**：
   - Skill 会自动检测 KAMI 排版工具是否已安装
   - **如果未安装**：Skill 会引导你一键安装 KAMI（提供安装命令和指导）
   - **如果已安装**：直接跳过安装步骤
   - 安装完成后，Skill 会引导你填写个人配置（重点客户、行业、区域等）

3. **首次完整演示**：配置完成后，Skill 会立即生成一份示例早报并应用 KAMI 排版，让你看到完整效果

4. **后续使用**：自动读取工作记忆中的配置，无需重复询问，直接生成 KAMI 排版的 PDF

**注意**：首次使用时会自动引导安装 KAMI，无需手动预装。如果后续排版时检测到未安装，也会再次提示安装命令。

### 触发方式

对 Aily 说以下任意一句即可触发：

- "帮我看看客户有什么新动态"
- "客户动态预警"
- "客户商机早报"
- "帮我监控客户动态"

### 输出结果

Skill 会自动完成以下流程：

1. 搜索并去重客户动态（Researcher）
2. 评估商机价值并排序（Analyst）
3. 生成顾问式跟进话术（Writer）
4. 质检话术质量（Evaluator）
5. 检查 KAMI 是否已安装（Formatter 前置检查）
6. 应用 KAMI 排版生成 PDF（Formatter）

最终输出一份符合印刷级设计规范的专业 PDF，包含：
- 暖米色底色 `#f5f4ed`
- 油墨蓝强调色 `#1B365D`
- 仓耳今楷02/Charter 衬线字体
- 高优先级商机突出标记
- 可点击的原文链接
- 完整的质检报告

**如果 KAMI 未安装**：Skill 会暂停并提示安装命令，安装完成后继续生成 PDF。

## 支持的动态类型

| 动态类型 | 有效期 | 商机切入角度 |
|---------|--------|-------------|
| 融资/收购/IPO | 7天 | 扩张期差旅量激增，支出管控窗口期 |
| 出海扩张 | 14天 | 海外商旅管理复杂度，多币种合规 |
| 产品发布/新品上市 | 14天 | 销售出差量翻倍，差旅管理跟不上 |
| 人事变动（CFO/财务/行政） | 14天 | 新领导上任90天管理升级窗口期 |
| 上市/IPO准备 | 14天 | 审计合规要求，费控体系搭建 |
| 战略合作/并购 | 14天 | 团队整合，支出管理统一化 |
| 获得奖项/荣誉 | 14天 | 品牌影响力提升，继续扩张 |

## 双轨话术体系

- **飞书融合版**：客户是飞书用户，强调飞书+分贝通一体化
- **独立版**：客户非飞书用户，强调分贝通独立价值
- 不确定时默认使用独立版

## 目标客户画像

- **规模**：200-5000人，销售/业务人员70人以上
- **差旅量**：年差旅支出300万-2000万
- **行业**：智能制造、生物医疗、传统制造、零售品牌、门店连锁、企业服务
- **排除**：国央企、体制内企业、电商、游戏行业
